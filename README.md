# Jukebox Real Music

Vanilla music discs that play classic Minecraft YouTube parodies.

Drop a disc in a jukebox and hear the full songs — Revenge, TNT, Fallen Kingdom, and more — with custom album art and in-game song titles. Works on Java Edition. No mods required.

## Download

**[Latest release](https://github.com/trolle6/JukeboxRealMusic/releases/latest)**

## Installation

1. Download the latest **`Jukebox-Real-Music-*.zip`** from [Releases](https://github.com/trolle6/JukeboxRealMusic/releases).
2. Unzip it into your Minecraft `resourcepacks` folder:
   - **Default launcher:** `%appdata%\.minecraft\resourcepacks\`
   - **Prism / MultiMC:** `<instance>/.minecraft/resourcepacks/`
3. In-game, open **Options → Resource Packs** and enable **Jukebox Real Music**.

The zip contains a folder named **Jukebox Real Music** — that's the resource pack.

**Cloning the repo?** Audio files are stored with [Git LFS](https://git-lfs.com). After cloning, run `git lfs pull`, then copy the `Jukebox Real Music` folder into `resourcepacks`.
