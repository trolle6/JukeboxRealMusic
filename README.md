# Jukebox Real Music

> Vanilla music discs, but they actually play classic Minecraft YouTube parodies.

A Minecraft Java Edition resource pack that replaces 13 vanilla music discs with full fan-made parody songs — Revenge, TNT, Fallen Kingdom, and more — plus custom album-art textures and jukebox song titles. No mods required.

**Quick facts:** 13 tracks · Java 1.19.4–1.21.x · resource pack only · ~60 MB

See [SUMMARY.md](SUMMARY.md) for a short project overview.

## Installation

1. Download the [latest release](https://github.com/trolle6/JukeboxRealMusic/releases) or clone this repository.
2. Copy the **`Jukebox Real Music`** folder into your Minecraft `resourcepacks` directory:
   - **Prism/MultiMC:** `<instance>/.minecraft/resourcepacks/`
   - **Default launcher:** `%appdata%\.minecraft\resourcepacks\`
3. In Minecraft: **Options → Resource Packs →** enable **Jukebox Real Music** and move it to the top.

If you downloaded the release `.zip`, unzip it into your `resourcepacks` folder — it already contains a folder named **Jukebox Real Music**.

## Compatibility

Works on **Minecraft Java Edition 1.19.4 through 1.21.x** (pack formats 13–75).

This pack only replaces music disc audio, textures, and names. It does not require mods. Any mod that adds custom discs will not be affected unless it reuses the same vanilla disc IDs.
